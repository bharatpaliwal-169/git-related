# git-related

### delete .git from folder
```
 rmdir -Force -Recurse .git
```
```
 rm -rf .git
```
### Check which branch

```
git checkout b_name
```

### new branch
```
git checkout -b b_name
```

### clone n pull
```
git clone [link]

git pull 

git pull origin [branch]

```

### update individual branch with main
```
 git checkout main
 git pull
 git checkout validator
 git merge main
 git push
```
